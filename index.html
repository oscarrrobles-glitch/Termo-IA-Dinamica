Nota para incrustar en Moodle:
  1. Habilita GitHub Pages (Settings → Pages) para el repositorio, selecciona la rama main y la carpeta / (root). Después de publicar, la URL será: https://oscarrrobles-glitch.github.io/Termo-IA-Dinamica/
  2. En Moodle puedes incrustar la página con un bloque HTML usando un iframe. Ejemplo: 
    <iframe src="https://oscarrrobles-glitch.github.io/Termo-IA-Dinamica/index.html" width="100%" height="700"></iframe>
  3. Si no habilitas Pages, la alternativa es usar el archivo crudo: https://raw.githubusercontent.com/oscarrrobles-glitch/Termo-IA-Dinamica/main/index.html — algunos servicios aplican restricciones de embedding sobre raw.githubusercontent.net, por eso Pages es la opción recomendada.
