Concepto (5.8). La desigualdad de Clausius, ∮δQ/T ≤ 0, conduce al balance de entropía: ΔS = ∫δQ/T + Sgen, con Sgen ≥ 0. La entropía generada mide la irreversibilidad del proceso: Sgen = 0 solo en el límite reversible y Sgen < 0 es imposible. Explore tres procesos típicamente irreversibles y observe que Sgen > 0 siempre, y que tiende a cero al acercar el proceso al límite reversible.

Concepto (5.9). Para un sistema cerrado: S₂ − S₁ = ∫δQ/Tb + Sgen. Para un volumen de control en estado estacionario: 0 = Σ Q̇k/Tk + ṁ(s₁ − s₂) + Ṡgen, es decir Ṡgen = ṁ(s₂ − s₁) − Q̇/Tb. El signo de Ṡgen clasifica el proceso: positivo → irreversible (posible), cero → reversible (límite), negativo → imposible.

Concepto (5.10). El proceso isentrópico (s = constante) es el modelo ideal de turbinas, compresores y toberas adiabáticos. La eficiencia isentrópica compara el desempeño real con ese límite: ηT = wreal/ws (turbina), ηC = ws/wreal (compresor), ηN = V₂²/V₂ₛ² (tobera).
